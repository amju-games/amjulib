Check out source and build
This project is in an svn repository on googlecode. Please email me with your google email address to be added to the project.

To check out the source code on Windows:

  • Install Tortoise SVN or your favourite SVN client
  • Create a google account if you don't have one
  • Email me your google email address to be added as a contributor
  • Check out: right-click in your projects directory, choose SVN checkout and check out amjulib, like this:



    To build the project:
  • Install Microsoft Visual Studio 2010
  • Within the source code you checked out, go to amjulib/Test/ve1/Script/Windows/ve1 and open ve1.sln

  • Build and run... it should work. Please contact me if not!

    To check out the source code on Mac OS X:
  • Use terminal or xterm: something like this: 
        cd ~/my_projects_dir
        svn co https://amjulib.googlecode.com/svn/trunk/ ./amjulib
  • Then to build it, well, you have to build the amjulib library, then the game itself: 
        cd amjulib/Script/MacOSX
        export AMJULIB_MAC_SCRIPT_PATH=`pwd`
        make
        cd ../../Test/ve1/Script/MacOSX
        make
        make release
        make run
